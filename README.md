# 🗡️ Fruit Assassin 🗡️
### egr425-project-3

> #### A Violent Twist on the Classic Game: Fruit Ninja!🍊
>
> - **M5Core2** can sabotage the player.
> - **Mobile Device** plays the classic game while trying to earn enough points to freeze the saboteur and win.
>
>  *Everything* is going according to **plan**.

> ##### **M5Core2** 💯
>
> - Can spawn in **bombs** to decrease the player's points if *sliced*.
> - Can trigger blindness for the player.
> - Gains the points that the player loses when they slice the bombs.
> - Gets points added every other second.
> - Can spend points to sabotage.
> - Wins if player's points reaches zero *or* the saboteur gains enough points themselves.

> ##### **Mobile Device** 📱
>
> - Can slice **fruits** to gain points.
> - Can use points to *freeze* the saboteur (saboteur cannot gain points or activate any sabotages).
> - Starts with a certain amount of points.
> - Wins if they gain a certain number of points.
